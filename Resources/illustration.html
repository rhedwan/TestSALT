Event plane

Event vertices are logical moments: ARRIVAL, END_SERVICE, CLOSE_DOOR. Self-loops mean the event re-schedules itself (e.g. each arrival schedules the next).

Object plane

Physical entities holding state: the scheduler (clock + FEL), four priority queues, a priority selector, and the single CPU server. Solid arrows are entity motion.

Schedule links

Dashed arrows cross the planes: when the server starts service it writes END_SERVICE to the FEL; when arrival fires it writes the next ARRIVAL. No polling — everything is scheduled.
